# Project Memory

## Core
Noir & Ember: premium cafe & fine dining site. Modern fine dining aesthetic — deep noir with warm gold accents, hushed and cinematic.
Colors: Noir #0A0A0A bg, Card #1A1A1A, Gold #B8935A (primary), Cream #F5F5F0 (foreground).
Typography: Playfair Display (serif headings, italic accents in gold), Inter (body).
Nav pages: Home, Menu, About, Reservations. No cart/checkout/products/wishlist.
Reservations form is a request — toast success only, no backend.
Imagery: Unsplash food/interior photography, always dark/warm-toned.
